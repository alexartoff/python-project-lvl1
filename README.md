### INTRO
Учебный проект **"Игры разума"** онлайн-школы [hexlet.io](https://ru.hexlet.io), состоящий из пяти консольных игр.   
Для установки необходим Python3.8+.   
Демонстрация установки и самих игр см.ниже.
